Balance the Forces Game

How to Play:

  1. The game starts automatically with a random Action Force (blue bar).
  2. Use the slider below to set the Counter Force (red bar).
  3. Watch the canvas visualization:
    - Bars extend right for positive forces, left for negative.
    - Net Force (bottom bar): Orange if unbalanced, green when zero.
  4. Goal: Adjust the slider so Counter Force = negative of Action Force.
    Make Net Force = 0 N to win!
  5. Click Start/Reset for a new challenge.
